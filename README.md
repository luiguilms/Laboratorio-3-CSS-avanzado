# Lab Flexbox – Semana 3
**Curso:** Hojas de Estilo en Cascada Avanzado · UTP 2026

Este proyecto es una práctica de diseño responsivo utilizando **CSS GRID**.

## Pasos para ejecutar

1. Abrir un terminal en la carpeta de su preferencia y ejecutar el comando: git clone https://github.com/luiguilms/Laboratorio-3-CSS-avanzado.git
2. Abrir la carpeta en VS Code.
3. Abrir index.html en el navegador (doble clic o con la extension Live Server de VSCode).
4. Reducir el ancho de la ventana para comprobar el diseño responsivo y los estilos GRID.
