Contenido Principal

Esta interfaz fue construida usando CSS Grid Layout. El contenedor principal define una grilla de 2 columnas y 3 filas, y cada bloque (header, sidebar, main, footer) ocupa su área asignada mediante grid-template-areas.

Elementos Grid

Son los hijos directos del contenedor y se posicionan con grid-column y grid-row.

Hijos

grid-template-areas

Permite nombrar zonas del layout con texto legible, haciendo el CSS fácil de mantener.

Semántico

gap

Define el espacio entre filas y columnas sin usar márgenes manuales en cada elemento.

Espaciado
fr

Unidad fr

1fr ocupa una fracción del espacio disponible de forma flexible y proporcional.

Flexible

Responsivo

Con @media queries el grid puede reorganizarse para adaptarse a pantallas pequeñas.

Adaptable
